Задача: Создать телеграмм-бота "Телефонный справочник с возможностями работы с информацией, и экспорта информации"

/start - начало работы телеграмм-бота

В справочнике пользователь может посмотреть:
1. Все контакты;
2. Найти контакт;
3. Добавить контакт;
4. Удалить контакт.
